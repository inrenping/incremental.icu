# incremental.icu 隐私政策

**更新日期：** 2026年6月22日  
**生效日期：** 2026年6月13日  

---

感谢您使用 incremental.icu（以下简称"本服务"）。本服务由个人开发者 **inrenping**（联系邮箱：<inrenping@gmail.com>）运营。

本政策将向您说明，当您访问和使用本服务时，我们如何收集、使用、存储和保护您的个人信息。**在使用本服务前，请您仔细阅读并充分理解本政策。当您通过注册或登录等主动确认方式表示同意后，本政策方才生效。**

如您对本政策有任何疑问，或需要行使您的个人信息相关权利，请随时通过 <inrenping@gmail.com> 与我们联系。

## 一、我们收集的个人信息

我们仅收集为实现本服务目的所必需的信息。根据您使用的具体功能，我们可能收集以下信息：

| 收集场景 | 信息类型 | 是否必需 | 收集目的 |
|---------|--------|--------|---------|
| 注册/登录 | 电子邮箱地址 | 是 | 创建账户、身份验证、联系您 |
| GitHub 登录 | GitHub 用户 ID、邮箱地址 | 是 | 提供第三方快捷登录 |
| Google 登录 | Google 用户 ID、邮箱地址 | 是 | 提供第三方快捷登录 |
| 连接 Garmin 平台 | Garmin 授权凭证（access token） | 是（使用同步功能时） | 调用 Garmin API 获取/推送运动记录 |
| 连接 Coros 平台 | Coros 授权凭证（access token） | 是（使用同步功能时） | 调用 Coros API 获取/推送运动记录 |
| 数据同步 | 运动记录数据（FIT 文件，含运动类型、时长、距离、心率等） | 是（使用同步功能时） | 跨平台同步运动记录 |
| 服务访问 | IP 地址、设备型号、操作系统、浏览器类型、访问时间、页面路径 | 否（自动收集） | 服务运行维护、安全防护、统计分析 |

**敏感个人信息说明：** 您通过 Garmin 和 Coros 平台同步的运动数据（如心率、运动轨迹等）可能构成敏感个人信息。**我们将仅用于您指定的平台间同步目的，不会用于其他任何用途。** 收集此类信息前，我们会通过本政策显著告知并征得您的明确同意。

## 二、Cookie 及同类技术的使用

为保障服务正常运行和优化您的使用体验，我们可能会使用 Cookie、本地存储及其他同类技术：

- **会话 Cookie：** 用于维持您的登录状态，在您关闭浏览器后即失效。
- **分析技术：** 我们使用 **Google Analytics** 收集匿名化的访问统计信息（包括页面浏览量、访问来源、浏览器信息等），以帮助我们了解服务使用情况并持续改进。Google Analytics 不会收集您的个人身份信息。

**您的选择：** 您可以通过浏览器设置禁用 Cookie，但可能影响部分功能的正常使用。关于 Google Analytics 的数据处理方式，请查阅 [Google 隐私政策](https://policies.google.com/privacy)。您也可以通过安装 [Google Analytics Opt-out 浏览器插件](https://tools.google.com/dlpage/gaoptout) 选择退出被追踪。

## 三、我们如何使用信息

我们收集的信息主要用于以下目的：

- **提供与维护服务：** 创建和管理您的账户，实现跨平台运动数据同步；
- **身份验证与安全：** 验证您的身份，防范欺诈和滥用行为，保障账户安全；
- **客户支持：** 回复您的咨询、处理您的问题或投诉；
- **服务优化：** 分析服务使用趋势，改进产品功能和用户体验；
- **合规与法律义务：** 履行法律法规要求的义务。

**我们不会将您的运动数据用于广告投放、用户画像分析或提供给任何广告商。**

## 四、信息的存储

### 存储地域

您的个人信息存储于 **中国境外** 的服务器（由 Vercel Inc. 提供云计算服务，服务器位于美国等地）。由于本服务的运营者与服务器均位于境外，您的个人信息将不可避免地被传输至境外。

**跨境传输的合规保障：** 我们将采取符合适用法律要求的保护措施（包括签署标准合同条款等），确保您的个人信息在境外获得不低于中国法律规定的保护水平。

### 存储期限

我们仅在实现本政策所述目的所必需的期限内保留您的个人信息，具体为：

- 账户信息：在您持有账户期间持续保留；
- 运动同步数据：在您持有账户期间持续保留，以便您随时查阅和使用；
- 日志与分析数据：不超过 **12 个月**。

当您注销账户或我们停止运营后，我们将删除或匿名化处理您的个人信息，但法律法规另有要求的除外。

## 五、信息的共享、转让与公开披露

### 共享与转让

- **原则：** 我们**不会**向第三方出售您的个人信息。我们**不会**主动向第三方共享或转让您的个人信息，除非征得您的明示同意。
- **第三方 API 服务：** 为实现跨平台数据同步功能，我们会将您的运动数据（FIT 文件）传输给您指定的 Garmin 或 Coros 平台。这些平台将按照其自身的隐私政策独立处理您的数据。**我们仅传输您主动发起的同步任务所涉及的数据，不会批量传输您的历史全量数据。**
- **业务变更：** 如发生合并、收购、资产转让等情形，我们将要求新的控制者继续履行本政策，或重新征得您的同意。

### 委托处理与第三方服务清单

本服务接入了以下第三方服务商，它们可能在提供服务过程中接触您的信息：

| 第三方 | 用途 | 涉及信息 | 隐私政策 |
|-------|------|---------|---------|
| Vercel Inc. | 云托管服务 | IP 地址、访问日志 | [Vercel 隐私政策](https://vercel.com/privacy) |
| Google Analytics | 访问统计分析 | 匿名化使用数据、IP 地址 | [Google 隐私政策](https://policies.google.com/privacy) |
| GitHub OAuth | 第三方登录 | GitHub 用户 ID、邮箱 | [GitHub 隐私政策](https://docs.github.com/privacy) |
| Google OAuth | 第三方登录 | Google 用户 ID、邮箱 | [Google 隐私政策](https://policies.google.com/privacy) |
| Garmin API | 运动数据同步 | 运动记录（FIT 文件） | [Garmin 隐私政策](https://www.garmin.com/privacy) |
| Coros API | 运动数据同步 | 运动记录（FIT 文件） | [Coros 隐私政策](https://www.coros.com/privacy) |

### 免授权例外

根据相关法律法规，在下列情形中，我们处理您的个人信息**无需另行征得您的同意**：

1. 为订立、履行您作为一方当事人的合同所必需；
2. 履行法定职责或法定义务所必需；
3. 为应对突发公共卫生事件，或为保护您或他人的生命、财产安全所必需；
4. 为公共利益实施新闻报道、舆论监督等行为在合理范围内处理个人信息；
5. 法律、行政法规规定的其他情形。

## 六、信息的保护

我们采取符合业界通行标准的安全措施保护您的个人信息：

- **传输加密：** 使用 SSL/TLS 加密技术保护数据传输过程的安全；
- **存储加密：** 对敏感信息进行加密存储；
- **访问控制：** 严格限制对用户数据的内部访问权限；
- **定期审查：** 定期审查安全实践，持续改进保护措施。

尽管我们采取上述措施，但请注意任何安全措施都无法做到绝对安全。**如因我们的过错导致您的个人信息泄露，我们将依法承担相应的法律责任。**

### 安全事件处置

如发生个人信息泄露、篡改、丢失等安全事件，我们将立即启动应急预案，并依法通过站内通知、电子邮件或公告等方式告知您：

- 安全事件的基本情况和可能的影响；
- 我们已采取或即将采取的处置措施；
- 您可采取的自主防范和降低风险的建议；
- 我们的联系方式。

同时，我们将依法向监管机构报告安全事件的处理情况。

## 七、您的权利

根据适用的个人信息保护法律，您享有以下权利：

### 1. 查询与更正

您可以在 **【账户设置】** 页面查看和修改您的基本资料。如需查询或更正其他个人信息，请联系我们。

### 2. 删除

在以下情形中，您可以请求删除您的个人信息：

- 处理目的已实现、无法实现或为实现处理目的不再必要；
- 我们停止提供服务，或者信息保存期限已届满；
- 您已撤回同意；
- 我们违反法律、行政法规或约定处理您的个人信息；
- 法律、行政法规规定的其他情形。

### 3. 撤回同意

您有权撤回已授予的同意。撤回不影响撤回前基于您同意已进行的个人信息处理活动的合法性。撤回同意后，您可能无法继续使用部分或全部服务。

### 4. 注销账户

您可以随时在 **【设置 > 注销账户】** 提交注销申请。我们将在 **15 个工作日内**完成审核和处理。

账户注销后，除法律法规另有规定外，我们将删除或匿名化处理您的全部个人信息，包括账户信息、绑定的第三方平台授权以及所有同步的运动数据。

### 5. 可携带权

在技术可行的前提下，您有权请求将您的个人信息转移至您指定的其他个人信息处理者。

### 6. 响应时限

您可以通过 **<inrenping@gmail.com>** 行使上述权利。我们将在 **15 个工作日内** 回复您的请求。如拒绝您的请求，将向您说明理由。

**如您对我们的处理不满意，您还可以向网信部门等监管机构进行投诉举报。**

我们不会主动收集未成年人的个人信息。如发现我们在未获监护人同意的情况下收集了未成年人的个人信息，请立即联系我们，我们将尽快删除相关数据。
